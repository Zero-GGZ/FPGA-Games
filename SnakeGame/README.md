## 文件与模块介绍

| 文件名（模块名）      | 功能                                                 |
| --------------------- | ---------------------------------------------------- |
| top_greedy_snake.v    | 顶层模块                                             |
| game_status_control.v | 游戏状态控制模块，根据游戏状态产生相应的控制信号     |
| apple_generator.v     | 苹果生成模块，在蛇吃到苹果后随机生成一个新的苹果坐标 |
| snake.v               | 蛇的运动模块                                         |
|                       |                                                      |
